A modern, animated, dark-themed portfolio website built with Next.js, React, TypeScript, and Tailwind CSS.

## 🚀 Features

- Beautiful animated dark background
- Responsive and mobile-friendly
- Animated skill icons and section transitions
- Glassmorphism contact form
- Contact info and GitHub link

## 🛠️ Built With

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)

## 📦 Getting Started

Clone the repo and run locally:

```bash
git clone https://github.com/HadiKarkoosh/portfolio.git
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
